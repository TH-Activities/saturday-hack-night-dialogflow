### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rijfas/saturday-hack-night-dialogflow.git
   ```
3. Install required packages
   ```sh
   pip install -r requirements.txt
   ```
4. Enter your API in `config.py`
   ```py
   API_URL = 'YOUR API URL'
   ```
